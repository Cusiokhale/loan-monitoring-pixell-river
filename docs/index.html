Loan Monitoring API (1.0.0)

Download OpenAPI specification:

API Support: support@loanmonitoring.com

API documentation for the Loan Monitoring application. This API provides endpoints for managing loan applications with role-based access control (user, officer, manager) and user administration.

Loans

Loan application management endpoints

Create a new loan application

Allows authenticated users with the 'user' role to submit a new loan application. The loan is created with a 'pending' status.

Authorizations:
bearerAuth
Request Body schema: application/json
required
amount
required
number >= 0

Loan amount requested

purpose
required
string non-empty

Purpose of the loan

Responses

API Server

http://localhost:3000/api/v1/loans

Get all loan applications

Retrieves a list of loan applications. Officers and managers can view all loans. Supports filtering by status and userId.

Authorizations:
bearerAuth
query Parameters
status
string
Enum: "pending" "under_review" "approved" "rejected"
Example: status=pending

Filter loans by status

userId
string
Example: userId=user456def

Filter loans by user ID

Responses

API Server

Review a loan application

Allows loan officers to review a pending loan application and move it to 'under_review' status. Officers can add review notes during this process.

bearerAuth
path Parameters
id
required
string

Request Body schema: application/json
